# AI-Agent-RAG-demo


how to use this :

1. Go to github codespace

2. setup this on bash :

curl -fsSL https://ollama.com/install.sh | sh
ollama serve &
ollama pull gemma3

cant install on bash please use ai help ollama : https://docs.ollama.com/quickstart

3. create new file in python named chat.py

4. ollama pull gemma3:1b 
   
5. run ollama run gemma3

6. to stop or delete ollama : pkill ollama

7. chat it on terminal
